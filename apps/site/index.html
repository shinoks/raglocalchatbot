Widget Sandbox

Try the embedded RAG assistant on a realistic landing page.

This page loads the production-style widget bundle from localhost:8080/widget.js and talks to the API on localhost:8000.

Site origin: http://localhost:3000 API: http://localhost:8000 Bundle: http://localhost:8080/widget.js

Suggested flow

How to test it

  1. Open the admin panel at localhost:4173 and upload a document.
  2. Wait until the document status turns ready.
  3. Come back here and open the widget in the bottom-right corner.
  4. Ask a question that should be answered from that uploaded document.

Grounding policy

The widget should refuse weak answers.

If the document evidence is missing or weak, the assistant is expected to answer that it does not know based on the uploaded documents.
